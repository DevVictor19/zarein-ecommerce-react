# Zarein ecommerce

## Webshop made in ReactJS

### Description:

Zarein is a clothing ecommerce. It has a cart sytem, a contact form, a exhibition carousel and some pages: contact, about and catalog.

### Used Technologies:

- Redux
- React Router V6
- React Transition Group
- Custom Hooks
- Hooks
